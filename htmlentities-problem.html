Symbols like á, é, í, ã, õ, ç will only appear correctly in the page if the file encoding matches the meta charset tag's declared encoding. This file is saved in UTF-8 (Unicode) and the meta charset tag specifies UTF-8 as the page's encoding.

If you are unsure of the encoding needed or if you want to use symbols not easily accessible using the keyboard, you can use HTML entities.

Entities start with an ampersand (&) and end with a semicolon ';'. For example, &copy; will output ©.

Examples:

  • ©
  • & : amp = ampersand
  • < : lt = less than
  • > : gt = greater than
  • á : letter 'a' with acute sign
  • ã : letter 'a' with tilde
  • ç : letter 'c' with cedil
  • ♥ : hearts
  • ♣ : clubs
